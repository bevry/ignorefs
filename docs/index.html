Is Ignored Path Check to see if a path, either a full path or basename, should be ignored

isIgnoredPath(path: string, opts: [object]): boolean
Parameters
path (string) a full path or basename of a file or directory
opts ([object]) configurations options
Name Description
opts.ignorePaths [(false | Array<string>)] an optional listing of full paths to ignore
opts.ignoreHiddenFiles [boolean] whether or not to ignore basenames beginning with a "." character
opts.ignoreCommonPatterns [boolean] if true, will check the path and basename of the path against https://github.com/bevry/ignorepatterns
opts.ignoreCustomPatterns [(false | RegExp)] if a regular expression, will test the regular expression against the path and basename of the path
Returns
boolean: whether or not the path should be ignored
